**Installation**

Required dependencies
---------------------

* python3 (python2 is not supported)
* numpy
* xarray
* matplotlib
* basemap
* scipy
* netcdf4
* mpl_toolkits
* dask 
* dask_jobqueue
* esmpy
* xesmf

Optional packages
-----------------
* palettable (for color options in visualization)
* pygrib (only if working with grib files)
